Welcome to the "11ty-playground", a web development project born from my journey in mastering Eleventy, JavaScript, HTML, and CSS (and GitHub). 11ty-playground uses custom Bootstrap, fontawesome and Sass.

It is designed with love as a simple starter for personal websites and blogs. If want to use 11ty and get going fast, you can do a lot with this template. Think of it as your virtual playground to express yourself, be it through a blog about the stuff you make (if you're into that like me) or just a simple website about a project of yours.

Oh, and as for documentation, well, it's in the works.
