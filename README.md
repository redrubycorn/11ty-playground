# 11ty-playground

Welcome to the "11ty-playground", a web development project born from my journey in mastering Eleventy, JavaScript, HTML, and CSS (and GitHub).

11ty-playground uses custom Bootstrap, fontawesome and Sass. It is designed with love for personal websites and blogs that want to use 11ty and get going fast. Think of it as your virtual playground to express yourself, be it through a blog about the stuff you make (if you're into that like me) or just a simple website about a project of yours.

Explore this template, crafted during my journey of learning and use it for your own projects if you like.
If you love it, leave a star on <a href="https://github.com/redrubycorn/11ty-playground">GitHub</a>.

Oh, and as for documentation, well, it's in the works.
